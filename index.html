HTML Introduction

What is HTML?

  • Standart for Hyper Text Markup Language
  • Describes the structure of a Web Pages
  • Consist of a series of elements
  • Elements tell the browser how to display the content
  • Elements concict on pieces label with content

Basic elements

  1. La etiqueta "DOCTYPE" define la declaracion de este documento como HTML
  2. La etiqueta "head" contiene infotmacion adicional acerca de la pagina HTML
  3. La etiqueta HTML es el elemnto principal de una pagina HTML
  4. La etiqueta "TITLE" especifica el titulo de la pagina web
  5. La etiqueta "body" especifica el curpo de la pagina web (todos los elementos visible)
  6. La etiqueta "h1" define un texto de tamaño titular
  7. La etiqueta "h2" define un texto de tamaño subtitular"
  8. La etiqueta "p" define el texto de tamaño parrafo

¿Que es un elemento HTML?

Un elemento HTML esta definido por una etiqueta de inicio,seguido de algun contenido y una etiqueta de finalizacion
  • <nombre etiqueta>: etiqueta de inicio
  • contenido:entre la etiqueta de inicio y la etiqueta de finalizacion
  • </nombreEtiqueta>:etiqueta de finalizacion
